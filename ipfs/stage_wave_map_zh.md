# 关卡与波次配置 (Stage & Wave Configuration)

> **版本:** 1.0  
> **描述:** 定义不同关卡区间内，每一关包含的怪物波次数量。

## 📊 详细数据表

| 关卡范围 (Stage Range) | 每关波次 (Waves) | 说明 (Notes) |
| :---: | :---: | :--- |
| **1 - 20** | **3** | 初期阶段，波次较少，战斗节奏快。 |
| **21 - 50** | **4** | 进阶阶段，波次增加，考验续航。 |
| **51 - 100** | **5** | 中期阶段，标准战斗长度。 |
| **101 - 99,999** | **10** | 后期/无尽阶段，高强度的持久战。 |
