# 技能效果映射表 (Skill Effect Map)

> **版本:** 1.0
> **描述:** 定义技能系统的基础常量枚举以及具体效果的索引 ID (Index)。

## 1. 基础枚举常量 (Base Enums)

这些常量定义了技能的基本属性框架。

| 类别 (Category) | 键 (Key) | 值 (Value) | 说明 |
| :--- | :--- | :---: | :--- |
| **伤害类型**<br>(Damage Type) | `none` | **0** | 无伤害 / 辅助 / 治疗 |
| | `physical` | **1** | 物理伤害 |
| | `magic` | **2** | 魔法 / 元素伤害 |
| | `true` | **99** | 真实伤害 (无视防御) |
| **触发模式**<br>(Activation Mode) | `active` | **0** | 主动技能 |
| | `passive` | **1** | 被动技能 |
| **消耗类型**<br>(Cost Type) | `none` | **0** | 无消耗 |
| | `mp` | **1** | 消耗法力值 |
| | `hp` | **2** | 消耗生命值 |
| **目标范围**<br>(Target Scope) | `self` | **0** | 自身 |
| | `enemy_1` | **1** | 单体敌人 |
| | `enemy_2` | **2** | 2个敌人 |
| | `enemy_3` | **3** | 3个敌人 |
| | `enemy_4` | **4** | 4个敌人 |
| | `enemy_all` | **99** | 全体敌人 (AOE) |

## 2. 效果索引表 (Effect Registry)

技能具体的 Buff / Debuff / 特殊机制通过 `Index` 关联。

### ❤️ 恢复与生存 (Recovery & Survival)
| Index | 属性 | 描述 |
| :---: | :---: | :--- |
| **100** | hp | 恢复 **X%** 的生命值。 |
| **110** | mp | 恢复 **X%** 的法力值。 |
| **2** | hp | **(Debuff)** 降低 **X%** 的治疗效果。 |

### ⚔️ 属性增益 (Stat Buffs)
*注：区分固定数值 (Flat) 与 百分比 (%) 加成。*

| Index | 属性 | 类型 | 描述 |
| :---: | :---: | :---: | :--- |
| **200** | ap (攻击力) | **百分比 %** | 增加 **X%** 攻击力。 |
| **201** | ap (攻击力) | **固定值** | 增加 **X** 点攻击力。 |
| **202** | dp (防御力) | **百分比 %** | 增加 **X%** 防御力。 |
| **203** | dp (防御力) | **固定值** | 增加 **X** 点防御力。 |
| **210** | hp (上限) | **固定值** | 增加 **X** 点最大生命值。 |
| **211** | hp (上限) | **百分比 %** | 增加 **X%** 最大生命值。 |
| **212** | mp (上限) | **固定值** | 增加 **X** 点最大法力值。 |
| **213** | mp (上限) | **百分比 %** | 增加 **X%** 最大法力值。 |

### 💰 资源获取 (Economy)
| Index | 属性 | 描述 |
| :---: | :---: | :--- |
| **120** | exp | 增加 **X%** 的经验获取。 |
| **130** | gold | 增加 **X%** 的金币获取。 |

### 🛡️ 特殊战斗机制 (Combat Mechanics)
| Index | 属性 | 描述 |
| :---: | :---: | :--- |
| **300** | reflect | **反伤**：反射敌人造成伤害的 **X%**。 |
